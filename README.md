# Technical Event Hub

## Demo
**Deployment Link:** https://nats167.github.io/Tech-event/countdown.html
## Overview

I built a multi-page website (didnt realise it was supposed to be a single page) for a college technical fest using HTML, CSS, and JavaScript. The website consists of a homepage with a live countdown timer, an event schedule page, and a registration page.

The countdown timer dynamically updates in real time and displays the time remaining until the fest(july 1). The schedule page uses a grid layout to show all events throughout the day. Users can click on any event to open a popup containing additional details such as the venue and event information, making the schedule more interactive and easier to navigate.
The registration page includes form validation using JavaScript to ensure that users enter valid information before submitting. Form submissions are logged to the browser console as required.

I focused on creating a clean, responsive design that works across different screen sizes while maintaining a modern and simple technical-fest aesthetic.

## Features

- Live countdown timer to the event date
- Interactive event schedule timeline
- Clickable event cards with popup details
- Registration form with input validation
- Responsive design for desktop and mobile devices
- Multi-page navigation system

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Pages

- **Home** – Countdown timer and fest overview
- **Schedule** – Interactive timeline of events
- **Register** – Registration form with validation
